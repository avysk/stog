title:Installation <->

To build Stog, you will need:

  • OCaml >= 3.12 installed (not tested with prior versions, but it should be ok),
  • Xmlm >= 1.0.2 (not tested with prior versions),
  • Xtmpl.

Get the sources from git repository: git clone https://github.com/zoggy/stog.git

Just type make && make install This will install stog in the same directory as ocamlc and libraries in `ocamlc -where`/stog/.
