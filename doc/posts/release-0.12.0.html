Release 0.12.0 of stog is available from the page.

Changes:

  • [fix] missing include in mk-stog-ocaml-session,
  • install default templates for pages and styles,
  • --mods adds a directory where to look for modules,
  • handling a file or a list of files instead of a directory for compiling single documents,
  • --tmpl now adds a directory to look for templates,
  • OCaml toplevel output now highlights code corresponding to warnings and errors; use highlight-locs="false" in <ocaml-eval ...> to disable; when enabled, use print-locs="true" in <ocaml-eval ...> to force printing also the warning and error locations. To highlight the code, <span class="warning-loc"> and <span class="error-loc"> nodes are added,
  • use more HTML blocks in output of <ocaml-eval>: <div class="stderr"/>, <div class="stdout"/>, <div class="toplevel-out"/>.
