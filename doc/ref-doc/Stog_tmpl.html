Default templates.
type contents = Stog_types.stog -> Xtmpl.tree 
val get_template_file : Stog_types.stog -> Stog_types.elt -> string -> string
get_template_file stog elt file returns absolute filename of the given template filename. If file is relative and implicit then it is concatenated to the stog template directory. If file is relative and not implicit (i.e. it starts with . or ..), it is concatenated to the element source file directory. Else (the filename is absolute), it is returned as is.
val read_template_file : Stog_types.stog -> Stog_types.elt -> ?raw:bool -> string -> Xtmpl.tree
read_template_file stog elt file returns the content of the given template filename, after calling Stog_tmpl.get_template_file to get the final filename.
raw : indicate whether to read the template as XML (false) or as CData (true). Default is false.
