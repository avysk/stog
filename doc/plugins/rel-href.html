By now, this plugin only adds a rewriting level to change URLs with fragments in href attributes of a document to keep only "#" followed by the fragment, when refering to the same document.

For example, in a document whose final URL is http://foo.bar/buz.html, href attributes containing http://foo.bar/buz.html#frag with be changed to refer to only #frag. This prevents some browsers from reloading the whole page when following a link to another part of the document.

Usage:

$ stog --package stog.rel-href ...
